# Phase 1 Roadmap: Foundation and Data Setup

Date: 2026-07-10

## Steps
1. Choose the server, UI, database, and AI service components for v0.1.
2. Set up the repository layout for frontend, backend, data, scripts, and tests.
3. Define the initial entities: course, video, chunk, and question log.
4. Decide the minimum metadata fields for each transcript entry.
5. Establish configuration patterns for local development and secrets.
6. Capture the chosen structure in the project planning documents.

## Output
A stable foundation for the ingestion, indexing, and retrieval work that follows.
